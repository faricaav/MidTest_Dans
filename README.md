# MidTest_Dans

### Clone Repo :
```sh
git clone https://github.com/faricaav/MidTest_Dans.git
```

### Backend
```sh
 cd backend
 ```
 1. Install Package
 ```sh
 npm install
 ```
 2. Start
 ```sh
 npm start
 ```
 
### Frontend
```sh
 cd frontend
 ```
1. Install Package
 ```sh
 npm install
 ```
2. Start
 ```sh
 npm start
 ```
